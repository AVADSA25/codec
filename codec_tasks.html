How Heartbeat Works

The heartbeat daemon runs in the background at your chosen interval. Each cycle it checks service health (LLM, Whisper, TTS, Vision, Dashboard), memory stats, pending tasks, and any alerts you configure below. Triggered alerts appear in the Reports tab and as dashboard notifications.

Alert types:
price — Track crypto prices via CoinGecko. Set asset, threshold %, direction (up/down/any)
email_check — Count unread emails in macOS Mail inbox
disk_usage — Warn when disk usage exceeds a threshold %
Toggle alerts on/off without deleting them. Disabled alerts are skipped each cycle.
